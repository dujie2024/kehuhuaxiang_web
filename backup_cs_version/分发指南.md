## ⚙️ 用户配置指南

### 1. 数据库配置

用户需要编辑 `config.ini` 文件:

```ini
[database.carpark]
host = 数据库服务器地址
user = 数据库用户名
password = 数据库密码
database = carpark
port = 3306

[database.dzwl]
host = 数据库服务器地址
user = 数据库用户名
password = 数据库密码
database = dzwl
port = 3306
```

### 2. 系统要求

**硬件要求**:
- CPU: 双核 2.0GHz 以上
- 内存: 4GB 以上
- 硬盘: 100MB 可用空间

**软件要求**:
- Windows 10/11 (64位)
- MySQL 5.7+ 或 8.0+
- 网络连接 (访问数据库)

### 3. 网络配置

确保以下端口可访问:
- MySQL 端口 (默认 3306)
- 数据库服务器网络可达

## 🔧 故障排除

### 常见问题

**Q: 程序无法启动**
A: 检查系统是否为 64位 Windows,关闭杀毒软件重试

**Q: 数据库连接失败**
A: 
- 检查 MySQL 服务是否启动
- 验证 config.ini 配置是否正确
- 确认网络连接正常

**Q: 界面显示异常**
A: 
- 更新显卡驱动
- 检查系统 DPI 设置
- 重启程序

**Q: 图表无法生成**
A: 
- 确保有足够内存
- 检查数据量是否过大
- 重启程序

### 日志查看

日志文件位置: `logs/` 目录下
- `app.log`: 主程序日志
- `models.database.log`: 数据库日志

### 性能优化

**大数据量查询**:
- 缩小日期范围
- 使用索引优化
- 定期清理日志

**内存使用**:
- 关闭不必要程序
- 增加虚拟内存
- 重启程序释放内存
