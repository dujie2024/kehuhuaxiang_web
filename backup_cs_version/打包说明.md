# 打包说明文档

## 📦 打包为可执行程序

本指南将帮助你将 Python 项目打包为 Windows 可执行程序 (.exe),便于分发和使用。

---

## 🎯 打包方式

### 方式一: 一键打包脚本 (推荐)

```bash
# 安装打包依赖
pip install -r requirements_build.txt

# 运行打包脚本
python build.py
```

**优点**:
- 自动化流程
- 包含所有必要配置
- 创建便携版
- 生成版本信息

---

### 方式二: 手动打包

```bash
# 1. 安装 PyInstaller
pip install pyinstaller

# 2. 基础打包
pyinstaller --onefile --windowed app.py

# 3. 包含配置文件
pyinstaller --onefile --windowed --add-data "config.ini:." app.py

# 4. 包含日志目录
pyinstaller --onefile --windowed --add-data "config.ini:." --add-data "logs:logs" app.py
```

## 🔧 打包配置详解

### spec 文件配置

打包脚本会自动生成 `新朝旅游客户画像分析系统.spec` 文件:

```python
a = Analysis(
    ['app.py'],
    datas=[
        ('config.ini', '.'),     # 配置文件
        ('logs', 'logs'),        # 日志目录
    ],
    hiddenimports=[             # 隐式导入
        'customtkinter',
        'tkcalendar',
        'matplotlib',
        # ... 更多依赖
    ],
)

exe = EXE(
    name='新朝旅游客户画像分析系统',
    console=False,              # 隐藏控制台窗口
    icon='icon.ico',            # 应用图标
    version='version_info.txt', # 版本信息
)
```

### 重要配置说明

**--onefile**: 打包为单个 exe 文件
**--windowed**: 无控制台窗口
**--add-data**: 包含数据文件
**--hidden-imports**: 隐式导入模块

## 🚀 使用说明

### 1. 直接运行 exe 文件

双击 `新朝旅游客户画像分析系统.exe` 即可启动程序

### 2. 配置数据库

编辑 `config.ini` 文件:

```ini
[database.carpark]
host = 192.168.0.51
user = root
password = root
database = carpark
port = 3306

[database.dzwl]
host = 192.168.0.51
user = root
password = root
database = dzwl
port = 3306
```

### 3. 功能模块

- 📊 **停车数据分析**: 分析停车场使用情况
- 🎫 **预订数据分析**: 分析票务预订数据
- 🔒 **电子围栏客流**: 分析景区进出客流

## 🐛 常见问题

### 1. 打包失败

**问题**: ModuleNotFoundError
**解决**: 在 hiddenimports 中添加缺失的模块

```python
hiddenimports=[
    '缺失的模块名',
    # ...
]
```

### 2. 程序无法启动

**问题**: 找不到配置文件
**解决**: 确保 config.ini 在 exe 同级目录

### 3. 界面显示异常

**问题**: 字体或图标缺失
**解决**: 检查 hiddenimports 中的 GUI 相关模块

### 4. 数据库连接失败

**问题**: pymysql 模块缺失
**解决**: 在 hiddenimports 中添加 'pymysql'
